# Kratos Agents Turbo

Sistema base para processamento jurídico em lote com:

- FastAPI
- Celery
- Redis
- Supabase
- skills locais
- agente jurídico simples
- servidor MCP simplificado

## Como rodar

1. Copie `.env.example` para `.env`
2. Suba os serviços com Docker Compose

docker compose up --build
